================================================================================
  QUICK START GUIDE
================================================================================

STEP 1: Install Dependencies (1 minute)
npm install

STEP 2: Setup Database (3 minutes)
mysql -u root -p < scripts/init-database.sql

STEP 3: Configure Environment (2 minutes)
Create .env.local with:
  DB_HOST=localhost
  DB_PORT=3306
  DB_USER=root
  DB_PASSWORD=root
  DB_NAME=property_tax_db
  GOOGLE_GENERATIVE_AI_API_KEY=your_key_from_ai.google.dev

STEP 4: Run Development Server (1 minute)
npm run dev

STEP 5: Open Application
Visit: http://localhost:3000

TOTAL TIME: ~10 minutes

For detailed setup, see: SETUP_GUIDE.md
For quick reference, see: QUICK_REFERENCE.md

================================================================================
  DATABASE SCHEMA
================================================================================

TABLE: properties (100 records)
├── property_id (INT, PK)
├── owner_name (VARCHAR)
├── ward (VARCHAR) - Ward 1-5
├── zone (VARCHAR) - A, B, C
├── property_address (VARCHAR)
├── tax_amount (DECIMAL)
├── due_amount (DECIMAL)
├── last_payment_date (DATE)
├── payment_status (ENUM: PAID, UNPAID, PARTIAL)
└── timestamps

INDEXES:
- idx_ward (ward)
- idx_zone (zone)
- idx_payment_status (payment_status)
- idx_due_amount (due_amount)

VIEWS:
- ward_collection_summary
- zone_collection_summary

SAMPLE DATA:
- Total Properties: 100
- Paid: 25 (~25%)
- Unpaid: 45 (~45%)
- Partial: 30 (~30%)
- Total Tax Amount: ~₹1,500,000
- Total Pending Tax: ~₹600,000
